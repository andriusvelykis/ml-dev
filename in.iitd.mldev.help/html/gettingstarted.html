Getting started

Creating a new project

Before you can write any SML program, you have to create the project that will contain it. A project is essentially a container for files and folders, and is used by Eclipse for organizing related resources together.

  1. To create a new project, select the menu item File > New > Project... to open the New Project wizard.
  2. Eclipse provides different kinds of projects, any of which can contain SML files. For now, select Simple > Project and click Next.

  3. In the next page, give a name for the project and click Finish.

The new project will appear in the Navigator view on the left. The Navigator allows you to manage all the projects, files and folders you create in Eclipse, which are collectively called the workspace.

Note: If you expand the project in the Navigator, you'll see a single file named .project. Don't do anything to this file; Eclipse uses it to store its information about the project.

Creating new files

Now you're ready to begin writing SML files in Eclipse. To get started, we'll create a new file.

  1. Right-click on the project and select New > File... to open the New File wizard.
  2. Enter a name for the file, and click Finish.

A new file will be created and will appear in the Package Explorer. The new, empty file will also be opened in an SML editor. Similarly, you can also create folders to organise files within a project.

Importing existing files into Eclipse

If you want to use Eclipse to work on existing files, you'll need to import them. This will copy the files into Eclipse's workspace so that they can be edited in Eclipse. (Note that any changes you make will not affect the original files.)

  1. Right-click on the project into which the files should be copied, and select Import...
  2. In the Import wizard, select File system and click Next.

  3. Click Browse... and choose the directory containing the files.
  4. The contents of the directory will be listed. Select the files that you want to import.

  5. Click Finish. Expand the selected project to see the imported files.

To get the files back out into the file system, use the analogous Export wizard.
